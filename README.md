## Overview
This project analyzes aviation bird strike incidents between 2000 and 2011 using real-world FAA data.  
The goal is to identify trends, high-risk phases of flight, impacted airports and airlines, and the financial impact of bird strikes through data visualization.

---

## Tools Used
- Python
- Jupyter Notebook
- Pandas, NumPy
- Matplotlib, Seaborn

---

## Key Analysis
- Year-wise trend of bird strike incidents
- Top U.S. airlines and airports affected
- Distribution across flight phases
- Altitude-based strike analysis
- Yearly cost impact of bird strikes
- Effect of pilot warning on strike severity

---

## How to Run
1. Clone the repository  
2. Open the Jupyter Notebook  
3. Run all cells
